## 正文（直接复制粘贴版）

家人们救命，最近 vibe coding 上头，副作用太真实——

跟 Claude 聊一下午，回头一看 docs 目录，**47 个 md 文件躺在那冲我笑** 🥲

PRD 改过三版、module spec 各自为政、plan 写完没人回看、还有一堆"先写下来回头看"的 memo……

最离谱的是 👉 **我自己都说不清这个项目到底做到哪了。**

装 Jira？太重。
开 Notion？还得复制粘贴一遍。
心累 ×100。

后来我索性自己写了个开源小工具——

✨ **docs-cockpit** ✨

扫一遍 docs 目录下所有 md，**自动生成一个 dashboard**（kanban + sprint 时间线 + 进度条），浏览器双击打开就能看。

- 零依赖、不联网、隐私安全
- 改 md → 重跑一行命令 → dashboard 自动同步
- Claude Code 用户更省事：一句"帮我把 docs 做成 dashboard"，它自己装、自己跑、自己出图

自用大半个月，**最大的改变不是工具多好用，而是我又能看清自己在做啥了。**

Python + MIT 协议，**完全免费开源**，谁用谁知道 🫶
